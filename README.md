# Bank Marketing Data Analysis

## Project Overview
This project analyzes a bank marketing dataset to understand customer behavior and identify factors influencing term deposit subscription.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
The dataset contains customer demographic, campaign, and economic information.

## Key Analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Visualization of customer trends

## Conclusion
The analysis shows that call duration and previous campaign outcomes strongly influence customer subscription decisions.
